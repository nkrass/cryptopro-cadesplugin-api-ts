# Upstream Script (Reference)

This repo targets compatibility with CryptoPro's upstream `cadesplugin_api.js` (`JSModuleVersion = "2.4.2"`).

Because redistribution rights for the upstream script may vary, the upstream file is **not committed** here by default.

## Fetch Locally

```bash
npm run upstream:fetch
```

This downloads the upstream script into:
- `upstream/cadesplugin_api.2.4.2.js`
- `upstream/cadesplugin_api.2.4.2.sha256`

## Official Sources

- Activation guide: `https://docs.cryptopro.ru/cades/plugin/plugin-activation`
- Script download: `https://www.cryptopro.ru/sites/default/files/products/cades/cadesplugin_api.js`
